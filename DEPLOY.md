# Как выложить голосование в интернет

## Вариант 1: Быстро поделиться ссылкой (пока комп включён)

1. Установи **ngrok**: https://ngrok.com/download  
2. Запусти проект: `python app.py`  
3. В другом терминале: `ngrok http 5000`  
4. Скопируй ссылку вида `https://xxxx.ngrok-free.app` — её можно скинуть людям.  

Минус: сайт работает только пока запущен `python app.py` и ngrok, и комп включён.

---

## Вариант 2: Бесплатный хостинг на Render.com (постоянный адрес)

1. Зарегистрируйся на https://render.com (можно через GitHub).  
2. Нажми **New → Web Service**.  
3. Подключи репозиторий с этим проектом (если кода ещё нет в Git — создай репозиторий на GitHub и залей проект).  
4. Настройки:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `gunicorn --bind 0.0.0.0:$PORT app:app`
5. Нажми **Create Web Service**.  
6. После деплоя получишь ссылку вида `https://твой-сервис.onrender.com`.

Важно: на бесплатном тарифе Render база SQLite сбрасывается при каждом перезапуске (раз в ~15 мин неактивности). Голоса будут сохраняться между заходами, но не навсегда. Для постоянного хранения позже можно подключить бесплатную PostgreSQL на Render и переписать код под неё.

## Вариант 3: PythonAnywhere (бесплатно, SQLite не сбрасывается)

1. Регистрация: https://www.pythonanywhere.com  
2. Вкладка **Web** → **Add a new web app** → Flask, выбери версию Python.  
3. Залей файлы проекта (через Git или загрузку).  
4. В настройках укажи путь к проекту и **WSGI file** (например, загрузи `app.py` и укажи в WSGI: `from app import app` и `application = app`).  
5. Укажи **Source code** и **Working directory** на папку с проектом.  
6. Перезагрузи приложение. Ссылка будет вида `https://твой-логин.pythonanywhere.com`.

На бесплатном тарифе сайт «засыпает» без посещений; первый заход после простоя может быть медленным.
